1. Зарегистрироваться в GitHub
2. Установить Git (консоль git)
3. Зарегистрироваться в Heroku при включенном  VPN - 1 раз при регистрации.
4. Установить терминал Heroku cli
https://devcenter.heroku.com/articles/getting-started-with-php#set-up скачать heroku.exe
5. Запустить heroku.exe
6. В консоли набрать heroku login - перенесет на страницу браузера с авторизацией
7. Создать новове приложение New App:  https://dashboard.heroku.com/new-app
8. Имя например php-ex-707
9. Создать Procfile
10. В нем vendor/bin/heroku-php-apache2 .
11. Создать файл index.php и в нем написать имя файла
<?php include_once("menu.html"); ?>
12. Создать файл с именем composer.json
в нем {}
13. зайти в консоль с папки с проектом
14. набрать heroku.com
15. Выбрать созданное приложение (php-ex-707)
16. Перейти в deploy
17. Набрать в консоли git init
18. Затем heroku git:remote -a php-ex-707
19. Затем
git add .
git commit -m "комментарий"
git push heroku master
20. Исправить ошибки 
 git config --global user.email "you@example.com"
 git config --global user.name "Your Name"
21. Еще раз набрать
git commit -m "комментарий"
git push heroku master
22. На сайте heroku зайти во вкладуку overview
23. Выбрать Configure Dynos
24. Перезугрузить страницу
25. Нажать на карандаш справа и confirm
26. Справа open app
27. Каждый раз при обновлении набирать
git add .
git commit -m "комментарий"
git push heroku master
28. Доступ к сайту 
https://php-ex-707.herokuapp.com/
